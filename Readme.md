# 🧠 Iyangtan

> A JARVIS-inspired AI assistant that combines voice interaction, RAG-based memory retrieval, and LLM-powered responses for personalized and context-aware assistance.

---

## 🚀 Overview

Iyangtan Echo is a voice-enabled AI assistant built using Python and Generative AI technologies. The assistant can understand natural voice commands, perform desktop-level tasks, remember past interactions, and generate intelligent responses using a Retrieval-Augmented Generation (RAG) pipeline.

The project focuses on combining voice interaction, intelligent memory, task automation, and real-time research capabilities into a single personalized AI assistant.

---

## ✨ Features

- 🎤 Voice-based interaction
- 🧠 RAG-based memory system using FAISS
- 💬 Context-aware conversational responses
- ⚡ Desktop task automation
- 🌐 Gemini integration for smart research & summarization
- 📊 Memory ranking using similarity, recency, and importance

---

## 🏗️ Tech Stack

- **Language:** Python
- **IDE:** PyCharm
- **LLM:** OpenAI / Gemini
- **Frameworks:** LangChain
- **Vector Database:** FAISS
- **Speech Recognition:** Whisper / SpeechRecognition
- **Text-to-Speech:** pyttsx3

---

## ⚙️ System Flow

```text
Voice Input
    ↓
Speech-to-Text
    ↓
Intent Detection
    ↓
Command Execution / RAG Retrieval
    ↓
LLM Response Generation
    ↓
Text-to-Speech Output
```
